# ABSA NextGen Wealth Studio

A financial planning MVP for young South African professionals — built with React + Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Stack

- React 18 + Vite
- React Router v6
- SARS 2024/25 tax calculations
- ZAR formatting & SA financial context (TFSA, RA, offshore allowances)
